# GDB
```
https://www.gnu.org/software/gdb/

GDB是GNU Project調試器，它使您可以查看另一個程序在“執行”期間正在執行的操作–或該程序崩潰時正在執行的操作。

GDB可以做四種主要的事情（以及支持這些事情的其他事情）來幫助您捕獲行為中的錯誤：
啟動您的程序，並指定可能影響其行為的所有內容。
使程序在指定條件下停止。
檢查程序停止時發生的情況。
更改程序中的內容，以便您可以嘗試糾正一個錯誤的影響，然後繼續學習另一個錯誤。
```
